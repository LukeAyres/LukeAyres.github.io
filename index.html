<Free Code Camp Projects/>

The projects below are all challenges provided by the website Free Code Camp. Each challenge provides an example website based on a certain theme (portfolio, quote-generator...) and the user must produce a similar website with original design features. The challenges come with a series of "user stories" that the student must pass to complete the challenge for example in this particular project the greetings section has to take up the entire screen of the user and there must be a projects section filled with several projects. You can find me on FreeCodeCamp here.
